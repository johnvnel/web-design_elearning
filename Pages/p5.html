Navigation bar and footer

The Navigation bar and the footer are two most essential elements on your webpage. The Nav bar will help you navigate through the content presented in your e-portfolio and the footer can have bookmarks or essential copyright information. In this section we will use bootstrap classes to add these components to our webpage. You can later change the color and position of these elements using CSS later on.

Use this link to view the Bootstrap documentation: Bootstrap Documentation

This folder will have the CSS stylesheets for most of the components you will need: Bootstrap Classes

Navbars

The navigation aspect of your website is the most pivotal part of your webpage. So don't hold back on making them look good. Take a look at these Bootstrap Navbar classes.

Footers

Making a footer using basic CSS shouln't be a hassle. If you want to try something new, take a look at these Bootstrap footer classes.
