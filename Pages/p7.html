Hosting your portfolio on GitHub pages

Now that you've set up your page, all you have to do it upload it on GitHub for people to view it. Follow the steps below to

  • Sign-up on GitHub
  • Create a new repository
  • Upload your files
  • Go to setting and click on pages which should be on the left side
  • Under source select the main branch and hit save
  • You should be given a link that you can share with others

Congrats, you've set up your e-portfolio!
